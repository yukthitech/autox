AutoX

What is AutoX

AutoX is a template based testing framework in which test cases are written in xml. It combines power of different frameworks like freemarker, jxpath, selenium etc and enables developers to take advantage of this in simple and intuitive way.

Why Autox

Platform Independent:

AutoX is written in Java (a platform independent language) which makes the framework also platform independent. Making test-suites executable in all Java supported operating systems.

Easy organization of test suites:

Test suite files can be easily organized in standard folder structure.

Embedded Template Support:

Embedded template support (using freemarker) is provided which makes defining queries, payloads etc in intuitive and simpler ways.

In built support for most common use cases:

In built support is provided for most common use cases involving SQL, REST, UI, files etc.

Extensive logging:

Extensive logs are generated by test case executions. Generates logs are searchable and formatted.

Easy Pluggability:

Easily plug custom codes and framework with simple code.
