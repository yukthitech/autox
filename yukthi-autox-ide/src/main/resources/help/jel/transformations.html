Transformation

Following special conversions are supported when standard json representation is not enough @transform can be used to transform the data into other format using any replacement expressions. In general this is used in conjunction with @value and falseValue (in conditions).

In these expressions current-value being transformed can be accessed as thisValue.

Using to convert result into json string. 
		{
			"result": {
				"@transform": "@expr: toJson(thisValue)",
				
				"@value": {
					"someVal": 1,
					"containers": "@fmarker: containers"
				}
			}
		}

The result will be something like below: 

		{
			"result": "{\"someVal\":1,\"containers\":[\"bank\",\"card\",\"insurance\"]}"
		}
