Object Param Type

Object is extension of Resource parameters. The specified resource should result in JSON content which will be parsed to Object.
Ohter than the types supported by resource, following extra types are supported:
Resource Type Example Description
ref ref:attr.emp Fetches specified property from context, and uses the result property-value as the value for param.
  {
   "name": "someName",
   "age": 30
}
 When no type is specified, the specified string value will be used input json source and parses it into object.


While loaind the source for the object to be loading, by default the data is loaded as map-of-maps (or lists). The data being loaded can be mapped to a specific bean type, this can be done by specifying data type at end after # (pound symbol).
   Example: res:/data/ext-data-provider.json#java.util.List <com.yukthitech.test.beans.TestCaseBean>
