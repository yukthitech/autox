UI Locator Param Type

UI locators are based on selenium and are useful in locating elements in various ways. Locator values should be of following format:
     <locatorType>:<locatorValue>

     Examples:
         id: button
         xpath: //input[@name='statusFld']

Following are different types of locators supported:
Locator Type Example Description
id id: button Locates the elements by using locator-value as element id.
class class: errElem Locates the elements by using locator-value as CSS class name.
tag tag: button Locates the elements by using locator-value as tag name.
name name: saveButton Locates the elements by using locator-value as element name.
css css: div>a Locates the elements by using locator-value as css expression.
xpath xpath: //input[@name='statusFld'] Locates the elements by using locator-value as xpath expression.
js js: div.red Locates the elements by using locator-value as jquery expression (can be used in webapplications which has jquery library).
