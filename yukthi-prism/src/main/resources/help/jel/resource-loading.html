Resource Loading

When complex content, like multi-line or double-quotes etc, needs to be added in json, the content can be placed in an external resource file and can be loaded using @resource.

@transform can be applied to transform loaded content into other format.

In Conjunction with @value 
		{
		  "DOC_TYPE": "TAX",
		  "CONTAINER_SUPRT_EN":  {
		     "@resource": "/doc-tax.xml",
		     "@transform": "@fmarker: normalizeXml(thisValue)"
		  }
		}
